# php version 切替

1. ターミナルにて
  > echo 'export PATH=/usr/local/opt/php@7.4/bin:$PATH' >> ~/.zshrc
  ＠7.４は切り替えたいversion
1. ターミナルにて
  > source ~/.zshrc
1. ターミナルにて
  > php -v
  versionが変更されていればOK
