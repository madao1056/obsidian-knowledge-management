# style.css（テーマを認識する為の必須ファイル）

```php
@charset "utf-8";
/*
Theme Name: テーマ名(例：junpeiblog)
Theme URI: テーマのホームサイトのURL
Description: テーマの説明
Author: 作者の名前
Author URI: 作者のサイトのURL
Version: バージョン
*/
```
とりあえずテーマ名があればOK（テーマ名は必須）
