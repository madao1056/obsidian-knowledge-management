# 告知文

## 6/30(金)
  ```plain text
【あなたのよしなに力を飛躍的に高める反則的サービス開始】
@everyone 

みなさま、お疲れさまです！
ぐっさん です！:muscle: 

長らく、お待たせしました！
『**しめサポ内 新サービス**』の説明会の告知です！！


新サービスの内容を超どストレートに言うと、

『ぐっさんが実務に伴奏してあなたの **よしなに力** を磨き上げます』

というサービスがこれから始まります！！


これまで70件以上の実務を通して、様々なよしなに対応をしてきました。
「よしなに」対応するスキル、いわゆる『よしなに力』は実務をやる上で絶対に必要です。
本当によしなに力がないとWeb制作は確実に生き抜くことができないと日々感じています。
さらにWeb制作だけでなく、『人』と接する全ての状況で活きてくる万能型のスキルになります。


しかし、実務を経験した方はわかると思いますが、
同じような案件であっても、制作会社が違えば、
やり取りや実装で指摘されるポイントって変わってきますよね。

そんな中、こんな不安を抱えながら実務をされているのではないでしょうか？

・実務全般で自分の考え方が合ってるか不安
・Web制作でのよしなに対応がわからない
・継続案件に繋がらない


このような不安があるのはあなただけではありません。


この度、そんな不安を払拭し、あなたの『よしなに力』を爆発的に伸ばす
サービスが誕生します！！

そして不安を払拭するだけでなく、
よしなに力をあなた自身で磨き続けるための考え方からお伝えすることで、
その後、自走しながらよしなに力を磨けるようになります！



私自身、よしなに力のおかげでお互いに良い関係性で付き合える会社さんや仲間に出会えました。
でも私の隣で伸び悩んでいる人がいるのも知っていました。
どうにかして悩んでいる人の力になれないか、私にできることってなんだろうと考えました。

そんな想いから、よしなに力講座を作り、
色々盛り込んだら合計9回という大ボリュームになりました。

このよしなに力講座を見て、早速実務に取り入れてくれた方も多くいます。
感想も多くいただき、本当に講座を作ってよかったなと思いました。
しかし、講座の内容だけでは不安が解消しきれていない方もいますよね。

そこで、私自身がサポーターという形をとることで、
直近であなたと向き合い、よしなに力を飛躍的に、爆発的に、
精神と時の部屋のように底上げをしていきたいと本気で思ってます。


これからフリーランスとして自由になっていくために、
必ず『よしなに力』を必要とする時が来ます。
その時が来てからでは遅いんです。



あなたの『よしなに力』で大切な『お相手』を喜ばせてみませんか？



説明会までに、よしなに力講座を必ず視聴してください。
一度見た方も再度、ご視聴をお願いします。
この講座はよしなに力を理解していただくための必須講座の位置付けになります。
ここを飛ばされてしまうと、サポートした際に認識のズレやミスマッチが起きてしまいます。

順を追って、私の方からこのチャットにリンクを載せていきます。
※来週月曜に第２〜4回を載せます。

▪️第1回よしなにの基礎理解講座〜よしなにって何？〜
https://youtu.be/3s5W2dMV-8k

第1回目はよしなに力をつけるための基本中の基本なるので、
絶対に落とせない内容になってます！

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

■日時：
7月7日（金）20時30分〜

■場所：
Zoomで行います！
直近になりましたら、Zoomリンクをお知らせします

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

せっかく実務ができるところまで来たのに、
ここで諦めて欲しくない、自信を持って対応してもらいたい。
私の経験、これからの試行錯誤を全てこのサービスに注いでフルコミットしていきます！
私は「やる」と決めたら「やる漢」です。

当日を楽しみにしています！
  ```
