# 最小値、基準値、最大値を設定

**clamp**というプロパティを使うと最小値、基準値、最大値を設定出来ます（IE非対応）
例えばフォントで使う場合、
```html
<p>フォント最小値、基準値、最大値を設定</p>
```
```css
p {
  font-size: clamp(16px, 3vw, 30px);
}
```
こうすると、
・基準は3vw
・画面幅を狭くしていくとフォントは小さくなっていくが、最小値は16px
・画面幅を広くしていくとフォントは大きくなっていくが、最大値は30px
という設定が可能。
