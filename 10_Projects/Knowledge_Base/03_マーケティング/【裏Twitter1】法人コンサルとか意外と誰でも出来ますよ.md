# 【裏Twitter1】法人コンサルとか意外と誰でも出来ますよ

```plain text
こんにちは、クラです。
無双ライティングマスタリーの特典である
僕が普段表向きのTwitterではあまり語れないような事をお伝えしていく
【裏Twitter】ですが早速第一ツイートしていきます。


アフターフォローメールという体ですが
もうこれだけで商品化した方がいいのでは？！って位の内容で配信していきます


配信頻度は3日に1通のスパンです。


1通のボリューム的にもこれ位のスパンが丁度良いと思います。


それでは…


「今回のテーマはネットビジネスの人って実店舗や法人コンサルしたがるよね」って話です


ネットビジネスだけじゃなくて
企業のコンサルティングとかも出来たらカッコいい！とか


実店舗にコンサルしてるって言ったら凄いと思われそうとか


まぁみんな色々と思いがあると思います。


ですが皆さん、実店舗のコンサルとか法人に対してのコンサルって意外と難しくないですし


ネットで20～30万とか売り上げた経験がある人なら全然実業などのコンサルも出来ると僕は思ってます。


結局の所これも知識持ってるか？持ってないか？なんですよ


例えば僕とかは実店舗を持った事無いですし、リアルビジネスもまだした事ありません。


でもコンサル入って売上を上げれる自信があります。てかなんならたまにやります


個人でお店やってる美味しいご飯屋さんとかに行った時に主人も話してる時にボソッと
「ここもっとこうした方が客入り良くなりますよ」とか結構言うんですが


大抵次行った時とかに超感謝されます。


リアルビジネスした経験もない
実店舗持った事ない若造に超感謝してるんです


これこそ知識の凄さです。
売上を上げる事が出来る知識さえ持っていれば相手が若造だろうが何者だろうが相手は感謝してくれるし、実際に売上を上げる事も可能です。


知識を知ってるってだけで
とんでもない希少な存在になるんですよ


例えば
もしあなたがジュエリー屋から売上を上げて欲しいと言うコンサル依頼があったらこんな話をすれば良いんですよ

＝＝＝＝＝＝＝＝＝＝＝＝
「まずいきなり初対面の人に数百万円もする宝石なんて売ってもまぁ成約率なんて低いんですよ」

「しかも宝石を買うなんていうのは一般庶民からしたら有り得ないので富裕層を狙わなきゃいけないです」

「ならまずは富裕層のリストを集める為に
新作のジュエリーを発表するという体のパーティーを開きましょう」

「もちろん無料で開催してシャンパンなんかも飲めるようにしましょう、そうすると富裕層達は知り合いも連れてきます」

「そうするとパーティーやシャンパン自体は赤字ですが富裕層のリストが集まります、そこに対してオファーをかけたり、後は不動産のとか節税のプロとかを紹介したりしても売上はバカスカ伸びますよ」

＝＝＝＝＝＝＝＝＝＝＝＝


これ妄想とかではなく
実際に富裕層のリストを取っていく方法として活用してアホほど儲けてる会社あります


しかもこれ気づいた方もいると思うんですが
全部ネットビジネスで身につける当たり前のマーケティング手法です。


・無料オファー(無料でパーティー)
・リストマーケティング(富裕層のリスト集め)
・クロスセル(不動産や節税のプロ)
etc...


これ見てる方は情報発信やコンテンツビジネスをされてる方が多いと思うので
結構ネットではやってる方いるんじゃないですか？


あのですね

皆さんが思ってるほど
実業の方ってマーケティング知らないです。


赤字でもいいからまずはリストを集めましょうなんて言っても「おい！赤字になってるじゃないか！どうなってるんだ！」ってなります


だから皆さんが当たり前にやってる事を少し抽象度上げてみて、飲食店だったらどうするかな？アパレルだったらどうするかな？と
考えていれば法人や実店舗へのコンサルなんて結構余裕で出来ますよ


ちなみにさっき話したジュエリー屋の話を
違うジュエリー屋にそっくりそのまま話してみるだけでも「コンサルしてくれ！」って言われる可能性大です


意外と今あなたがやってる事っていうのは抽象度を上げると実業にもいくらでも応用が効くんだよって話でした。


広い視点を持って生活してみましょう


それでは
```
